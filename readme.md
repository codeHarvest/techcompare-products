# TechCompare - Product Comparison App

A modern, responsive web application for comparing tech products across different categories. Built with Next.js, TypeScript, and Tailwind CSS, featuring an intuitive interface for side-by-side product comparisons.

- **Product Browsing**: Browse through a curated collection of tech products
- **Category Filtering**: Filter products by categories (smartphones, laptops, headphones)
- **Search Functionality**: Search products by name, brand, or features
- **Product Comparison**: Compare up to 4 products side-by-side
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Dark Mode**: Toggle between light and dark themes
- **Modern UI**: Clean, professional interface with smooth animations
- **Persistent State**: Comparison selections persist across page refreshes
